About This Project

Our Mission

The Student Registration Portal is a demo project developed during the Python Development Internship. Its goal is to demonstrate core Flask concepts such as routing, templates, forms, and flash messaging through a simple, real-world use case — registering and viewing student records.

Technologies Used

  • Flask
  • HTML
  • CSS

Project Highlights

  • No database — data stored temporarily in a Python list
  • Flash messages for user feedback
  • Responsive UI for mobile and desktop
  • Custom 404 error page
