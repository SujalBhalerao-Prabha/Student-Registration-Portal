About This Project

This portal allows students to register their details including Name, Email, Mobile Number, and Course. All registered students can be viewed on the Students page. This is a demo project built using Flask, HTML, CSS, Flash Messages, Templates, and Routing — with no database, data is stored temporarily in a Python list.
